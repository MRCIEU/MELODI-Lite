Previously we created MELODI, a method and tool to derive overlapping enriched literature elements connecting two biomedical terms, e.g. an exposure and a disease, (Elsworth et al., 2018).
Here, we present MELODI Presto. A quicker and more agile method to identify overlapping elements between any number of exposures and outcomes.
Access via the Web App, API or Notebook. Code is available on GitHub.
